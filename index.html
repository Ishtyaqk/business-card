Connect Supabase

Supabase is a free database. It stores your cards so they work on any device, anywhere. Setup takes ~2 minutes.

Step 1

Create a free Supabase project

Go to supabase.com → Sign up → New Project. Takes 1 minute.

Step 2

Run this SQL to create the cards table

In Supabase → SQL Editor → paste and run:

create table if not exists cards ( id bigint generated always as identity primary key, handle text unique not null, color text default '#0e0e0e', avatar text, fname text, lname text, title text, company text, bio text, phone text, email text, website text, linkedin text, twitter text, created_at timestamptz default now() ); -- Allow anyone to read cards (public profiles) alter table cards enable row level security; create policy "Public read" on cards for select using (true); create policy "Public insert" on cards for insert with check (true); create policy "Public update" on cards for update using (true); create policy "Public delete" on cards for delete using (true);
Step 3

Paste your Supabase credentials below

Project Settings → API → copy Project URL and anon public key.
